# 💕 Love Proposal Web App

A beautiful, interactive romantic proposal web application designed for mobile devices. Share this special moment with your loved one!

## 🌸 Features

- **7 Interactive Screens** - A journey from opening to celebration
- **Mobile-First Design** - Optimized for all mobile devices
- **Cute Cat Images** - Adorable cat avatars throughout
- **Animated Background** - Floating hearts, sparkles, and particles
- **Confetti Celebration** - Special effects when they say yes!
- **Tricky Button** - Fun "Let me think" button that moves away

## 📱 Screens

1. **Opening** - Cute cat introduction with "Tap to Begin"
2. **Trust Question** - "Do you trust me?" 
3. **Proposal** - "Will you be mine forever?"
4. **Guilt Modal** - Playful persuasion popup
5. **Photo Gallery** - Memory carousel with cute images
6. **Love Letter** - Heartfelt message
7. **Celebration** - Confetti and floating hearts

## 🛠️ Technologies

- **HTML5** - Structure and content
- **CSS3** - Styling, animations, and effects
- **JavaScript** - Interactivity and screen navigation
- **No frameworks** - Pure vanilla code, lightweight and fast

## 🚀 How to Use

1. Open `index.html` in any mobile browser
2. Share the link with your special someone
3. Watch them go through the romantic journey
4. Celebrate when they say YES! 🎉

## 🎨 Customization

- **Colors**: Edit CSS variables in `:root` section of `style.css`
- **Messages**: Modify text content in `index.html`
- **Images**: Replace files in `images/` folder
- **Timing**: Adjust animation durations in `script.js`
